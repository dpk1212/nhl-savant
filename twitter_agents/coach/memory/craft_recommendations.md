## 26–30 — Public-copy translation (expert 2026-07-12)

## 31–38 — Engagement / virality (expert round 2 — 2026-07-12)
31. **Open loop or die.** Audience must close the tension in replies. If copy already resolved it, the CTA is theater.
32. **No fake-balance.** One-sided proof → don’t ask fade/ride on that outcome. Ask sizing, philosophy, “would you have caught this,” or land a quotable agreement line.

35. **Philosophical camps > single-game strawmen** for reply physics.
36. **Clock/stakes** when live — timeless = scroll-past.
37. **One quotable line** that works extracted from context.

## 39–46 — Four-beat force (expert round 3 — 2026-07-12)
39. **Weight ≠ word count.** Brief that reads thin = missing beats, not missing adjectives.
40. **Four beats:** confront (“you”/interrupt) → confession or curiosity gap → data → consequence + clock.
41. **Soft statement closers die** — “you already know…” invites a nod; force a question + on-record stake.
42. **One tension axis** — dog-tribe OR sizing, not both stacked.
43. **Symmetrical camp proof** — both sides get $ / zero against / timing weight.
44. **Scroll-stop in first ~5 words** — callout, accusation, confession, or shocking number — not calm context.
45. **Mid-post pivot** required — one register change so the eye stops.
46. **Consequence line** — “pick wrong / we’ll see tonight / before either game starts.”

## 47–54 — Precision band (expert final — 2026-07-12)
47. **Single-play = 120–200 chars.** Comparison/binary ≤280. Every line earns a NEW fact/beat/tension or it dies.
48. **≤1 functional emoji** (📊/🔒). Decorative 🚨🔥💰 undercuts sharp brand.
49. **Colons + periods** over semicolons / excess em-dashes in data lines.
50. **Cut restates** — two sentences, one point = cut the second (tweet-1 “I get it…”).
51. **Public fights only** — no insider “is 2.5u too heavy?” to strangers.
52. **Accusations need teeth** — WR / $ / behavior backup in-post.
53. **Dead matchup tags die** — MIL@PIT without clock = cut.
54. **No invented injury/weather why** — gesture only at observable signal (early lock, unanimous, size).
