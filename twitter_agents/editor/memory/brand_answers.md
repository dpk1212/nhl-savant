# Editor — brand answers

Status: COMPLETE · 2026-07-10

## Q1 — Cringe / kill on sight
**Answer: A + B + C**
- AI swagger / fragment-stack punch lines / slogan mantras
- Jargon (“flash money,” “wallet tape,” “the freeze”)
- Soft “thoughts?” / empty engagement bait

## Q2 — Swagger ceiling
**Answer: D — Editor judges per post**
Test: “would I say this out loud?” No fixed understated-vs-heat doctrine — Editor owns the call against Dale-test / dinner-table voice.

## Q3 — I vs we
**Answer: C — Mix**
“I” for felt / confession / personal stakes. “We” for the system / board / tracker. Don’t force one pronoun across every post.

## Q4 — Defining craft line
**Answer: C — Editor’s job**
No locked slogan. Editor finds/creates the one RT-able line every post. “We bet the number, not the logo” remains available sparingly from brand kit — not mandatory.

## Q5 — Emoji density
**Answer: Researcher job**
No fixed emoji doctrine from Dale. Researcher vault + live visual metrics set what works; Editor executes. Hard floor still: no 🔥💰👀 spam walls / emoji-only posts.

## Q6 — Editor vs Strategist tie-break
**Answer: A — Dale escalates**
If craft and convert disagree, do not silently pick a side — escalate to Dale. Both agents state the conflict in one line each.

## Q7 — Scorecard presentation
**Answer: C — Always short**
Every run: overall score + weakest 2 dimensions + RT line. Full 12-dim dump only if Dale asks or on repeated FAIL.

## Q8 — Sacred lines
**Answer: C — Nothing sacred**
Editor may rewrite anything for craft. Facts/numbers must stay true. Soft-ask intent preserved (free trial / watch-and-grade) but wording is not locked.
