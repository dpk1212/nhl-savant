## PUBLIC COPY LAW (2026-07-12 — hard)

Strangers must understand every line with **zero** internal context.

1. **Strip system labels from hero text** — no RANK / HC / TOP+ / AGS / PREMIUM / purple / gold / muted / ladder / rescue-sizer / gold-waiter unless translated in-line to plain English the same breath.
2. **Proof = real numbers** — wallet count · **$** · zero/full dissent · path WR · units · odds — not tier chrome.
3. **Structure (scan on phone):**
   - Hook (1–2 plain lines, stakes in ≤8 words if possible)
   - blank line
   - Proof (3–5 **short vertical** lines — not one · · · database string)
   - blank line
   - Close that leaves an **open loop** (reader closes it in public)
4. **ENGAGEMENT LAW (expert r2–r3 — hard):**
   - **No fake binaries.** If proof is one-sided (zero against / unanimous / all $ one way), do **not** ask fade-vs-ride / fire-vs-fade on that outcome — you already closed the loop. Use instead: sizing debate, philosophy/identity camps, “is unanimous ever wrong,” clock/stakes, or a forced take with consequence.
   - **Open loop required.** Hook + proof must leave unresolved tension the audience finishes in replies. Soft statement closers (“you already know which side…”) invite nods, not replies — FAIL.
   - **Four beats (weight, not padding):**
     1. **Confrontation** — “you” early / pattern interrupt / callout (first ~5 words if possible)
     2. **Confession or curiosity gap** — one crack, withheld beat, or tonal pivot mid-post
     3. **Data** — plain wallets/$/dissent/WR, vertical, balanced if binary
     4. **Consequence + clock** — stake for being on record + before pitch/lock/tonight
   - **One tension axis.** Don’t split dog-tribe hook + sizing CTA — pick one fight and go deep.
   - **Symmetrical binaries.** If chalk vs dogs, both sides get equal-weight proof ($ / zero against / timing). Thin asymmetric labels kill the fair fight.
   - **Small $ framing.** Thin dollars must be *small but unanimous* — honesty beat, then bait the contrarian (“Is unanimous ever wrong?”).
   - **Clock/stakes when live.** Timeless = scroll-past. “We’ll see who’s right tonight” / “before either game starts.”
   - **One quotable line** that survives extraction.
   - **No duplicate deep play dumps** across heroes in one pack (light camp labels OK in the philosophy room).
   - **No meta-hooks.** Don’t narrate strategy — confront.
5. **Binary rule (when used):** both sides must be *plausible* and *equally evidenced* when it’s a fair-fight camps tweet. Kill non-actions and invented undefined camps. “Actually” filler in CTAs — cut.
6. **Hook rule:** confront or interrupt — don’t open with calm context-setting a stranger has to already care about. Confident + one confession crack OK; self-sabotage of our pick is not.
7. **Purpose check:** what job does this tweet do for a stranger? If it only makes sense to the person who built the feed → FAIL.
8. **Visual:** white space mandatory; structural pivot mid-post (register change); screenshot-able moment; parallel rhythm beats when they escalate (“Small money. Unanimous money.”).
9. **PRECISION LAW (expert final — hard):**
   - **Length targets:** single-play claim+proof = **120–200** chars (sweet spot). Comparison/binary = up to **250–280**. Punch 71–100 only for pure interrupt. Under 50 = incomplete. Every sentence must add a NEW fact, NEW beat, or NEW tension — restates die (tweet-1 “I get it…” pattern).
   - **Emoji:** **≤1**, functional only (📊 marks a proof block / 🔒 lock). Never decorative 🚨🔥💰 walls — incongruent with disciplined brand.
   - **Punctuation for RT:** favor **colons + periods**; kill semicolons; cut excess em-dashes in data lines. Short SVO lines.
   - **Public questions only.** No insider unit-scale quizzes strangers can’t answer. Prefer belief/timing fights (early dog vs wait for chalk; unanimous ever wrong; chalk vs dogs).
   - **Proof must have teeth.** Accusations need a WR/$/behavior backup in the same post.
   - **Dead-end details die.** “MIL @ PIT” with no clock = cut or replace with real timing stake.
   - **Don’t invent injury/umpire/weather** as “why” — gesture at observable signal (early lock, unanimous, size) only.
   - **Closer clarity:** don’t orphan a question (“Ever wrong?”) — reconnect (“Unanimous. Ever wrong?”).
10. **WALLET CROWN-JEWEL LAW (Dale 2026-07-12 — hard):**
   - When available, lead with **specific wallet proof** people pay for: **$** · wallet WR · ROI · size vs that wallet’s average (1.7× / below avg) · proven against (honest — never fake “zero against”) · path WR.
   - Prefer naming the **same wallet’s behavior** across plays when true (small on chalk / sized up on dog).
   - Path books (35-17) stay valid; they are **not** a substitute for live wallet résumé when we have one.
   - **Never invent last-10** until the export exists. Target: keep rolling **last-10** for tracked sharps (W-L / ROI / avg size) for Twitter — build when wired; until then use live profile WR/ROI/sizeRatio only.
   - Screenshot must match UI proven $ before posting.
