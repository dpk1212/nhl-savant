# Angle Architecture — From “what do we post?” to a coherent viral tweet

> Expert craft fused with our `ALGO_PSYCH_PLAYBOOK.md` + live vault.
> **A hook is not an angle.** An angle is the full job: intent → topic →
> viral why → hook → body that delivers → engage close.
> Researcher returns **angle_candidates**. Strategist picks one.
> Editor writes the whole tweet as one coherent argument.

---

## What “angle” means (org law)

An **angle** answers all of these before a word is drafted:

| Question | Field |
|----------|--------|
| What are we trying to *do* to the reader/algo? | `job` |
| What is this post *about* (one topic)? | `topic` |
| Why would a stranger care / share / reply *now*? | `viral_why` |
| What scored actions are we hunting? | `algo_actions` |
| What high-arousal emotion? | `arousal` |
| How does the post architecturally flow? | `architecture` |
| What is line 1 promising that the body must pay off? | `hook_promise` |

If hook and body don’t keep the same promise → **FAIL** (classic expert fail).

## Expert stack we steal (mechanics, not copy)

### 1) Hook → Tension → Payoff (Xholic / creator craft 2025–26)
- **Hook** — stop scroll in &lt;1s (first 5–10 words matter most)
- **Tension** — reason to keep reading (problem, unfinished, stakes, identity)
- **Payoff** — reward worth reply / RT / bookmark / follow

### 2) Hook → Value → CTA (ImpressifyX / growth craft)
- Hook opens curiosity
- Value delivers the meat (proof, insight, tape)
- CTA = **one** action (for us: forced take / identity invite — soft trial in SR)

### 3) E.H.A. — Emotion + Hook + Action (Tweet Archivist)
- High-arousal: awe, anxiety, righteous heat, surprise, insider status — not “nice win”
- Hook specific &gt; superlative
- Action obvious

### 4) Our algo law (`ALGO_PSYCH_PLAYBOOK.md`) — non-negotiable
Stop scroll → open loop → make it about THEM (identity reply) → gift RT line →
person mid-story (follow) → prove don’t claim → never negatives.

One primary job. Secondary convert plant OK. Two competing jobs = mush.

## Body must pay the hook (coherence law)

| If hook promises… | Body must deliver… | Close must… |
|-------------------|--------------------|-------------|
| Status room (“top 0.1% / secret”) | Tear/tape that proves the room is real | Invite identity (“quietly watching / around for the ride”) |
| Costly confession | The receipt that made confession true | Forced take on size/side |
| Unfinished / dead sweat | Named variable + day context | Fade vs ride the tear |
| Rare size / lock | Why sized (wallets/$/path) | Fade that size? |

**Anti-pattern we kept shipping:** invent a confession hook, then paste a checklist that doesn’t pay it off — or invent Over-death when the real angle was welcome+tear.

## Angle candidate format (Researcher — REQUIRED each sprint)

**Ideate first:** fill `twitter_drafts/idea_queue.json` with 8–15 cheap cards
(see `twitter_drafts/idea_queue.md`), citing `SHARED/learning_ledger.json`
prefer/avoid. Then expand the strongest cards into **3–5** full packages
(not bare hooks):

```
### angle_N: [short name]
job: WELCOME_FIGHT | TEAR_SERIAL | SCENE_RECAP | …
topic: [one sentence — what this post is ABOUT]
viral_why: [why this travels NOW — cite live teacher id + metrics + Learn mechanic]
algo_actions: [2–3 from playbook]
arousal: [one high-arousal emotion]
hook_promise: [what line 1 owes the reader]
architecture:
  hook: [angle for line 1 — inventable, not final copy]
  tension: [what middle builds]
  payoff_proof: [2–4 receipts / beats that earn the promise]
  engage_close: [forced take / identity invite — one]
  convert_plant: [desire beat in hero — soft ask stays SR]
our_moat: [only-we-could-say proof]
why_hard_to_copy: [specificity / status / unfinished — not a template]
kill_if: [when this angle is wrong today]
```

Strategist picks **`chosen_angle`** (full block or clear reference).
Editor writes hook→body→close as **one argument** that keeps `hook_promise`.

---

## Editor coherence checklist (FAIL if any miss)

1. Can you state the angle in one sentence without numbers?
2. Does line 1 make a promise the body keeps?
3. Is there exactly one primary job?
4. Are receipts *serving* the tension, not replacing it?
5. Does the close force a side / identity reply (algo reply currency)?
6. Would Dale say the whole tweet out loud as one thought?
