# Open Graph Social Sharing Image

## Required
Create an image named `og-image.png` in this `/public` folder.

## Specifications
- **Size**: 1200 x 630 pixels (required by Facebook/Twitter)
- **Format**: PNG or JPG
- **File size**: Under 8MB (ideally under 1MB)

## Suggested Design
Create a dark-themed image with:

### Text Content
- **Headline**: "NHL SAVANT"
- **Tagline**: "Advanced NHL Betting Analytics"
- **Key Stats** (update these with current performance):
  - 52.9% Win Rate
  - +30.3% ROI
  - $1,516 Profit

### Visual Elements
- Dark navy/black background (#0A0E27)
- Gold accents (#D4AF37)
- Hockey-related subtle graphics (puck, ice, etc.)
- Clean, professional typography

## Tools You Can Use
- **Canva**: Use their "Twitter Post" or "Facebook Post" template
- **Figma**: Create custom design
- **Photoshop/GIMP**: Full control
- **Online generators**: Search "open graph image generator"

## Quick Option
Visit [og-image-generator.vercel.app](https://og-image-generator.vercel.app) for a quick template-based solution.

## Once Created
1. Save as `og-image.png` in `/public` folder
2. Run `npm run deploy`
3. Test the preview at [metatags.io](https://metatags.io) or share the link on social media
