## Rules for the Content Loop agent
- Every post must include one standalone repost line in the first 2 lines, under 140 chars.
- At least one daily option must be a true thread: hook tweet + 2-4 evidence replies + final pick/choice.
- Use `pick_card`, record-flex, or sharp/public map formats before any recap-only format.
- Keep picks and units visible; support them with proven winners, HC sharps, dollars, percent sharp money, and Pinnacle when available.
- Use emoji on pick/recap posts to move closer to the 75% niche benchmark; default to alert/checkmark/baseball-style emoji, not random clutter.
- Close with either an A/B choice or one explicit repost ask; do not stack multiple CTAs.
- If no lock qualifies, post the pass as a useful threshold lesson, not an apologetic "no play" note.
- Never output contextless vent replies or bottom-post style complaints as standalone content.

## Anti-patterns checklist
- No post without a repostable line.
- No "Which ticket?" if the post already has a better A/B choice.
- No recap made only of checkmarks.
- No single-tweet wall when the evidence needs a thread.
- No "no lock" post unless it teaches the gate that failed.
- No invented stats, no copied niche wording, no sterile analyst voice.
